# proiect-de-pus-pe-rotom
Viitor dispozitiv ce foloseste un ESP32 pentru a detecta si a mentine o mica baza de date a ingredientelor de gatit din casa.\n
V-a integra TheMealDB API pentru a da retete pe care le poti gati cu ingredientele din baza de date.\n
O sa aiba un ecram cu touch (nu am nici cea mai mica idee cum o sa fac GUI-ul).
